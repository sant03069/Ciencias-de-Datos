# Actividad 1.1 – Fundamentos de Ciencia de Datos??

## ¿Qué es la ciencia de datos?

La ciencia de datos es una disciplina interdisciplinaria que se encarga de recolectar, procesar, analizar e interpretar grandes volúmenes de datos con el objetivo de obtener información útil y apoyar la toma de decisiones. Combina conocimientos de matemáticas, estadística, programación y análisis del negocio para transformar datos en conocimiento.

En la actualidad, la ciencia de datos es fundamental para empresas y organizaciones, ya que les permite entender mejor a sus clientes, optimizar procesos, predecir tendencias y mejorar su competitividad en el mercado.

## Componentes principales de la ciencia de datos:

**-Datos:**

Son la materia prima. Pueden provenir de diferentes fuentes como ventas, redes sociales, sensores, páginas web, entre otros. Sin datos, no existe análisis.

**-Estadística y matemáticas:**

Permiten analizar los datos, encontrar patrones, relaciones y realizar predicciones. Son esenciales para interpretar correctamente la información.

**-Programación:**

Se utilizan lenguajes como Python o R para manipular, limpiar y analizar grandes volúmenes de datos de manera eficiente.

**-Visualización de datos:**

Consiste en representar la información mediante gráficos, tablas o dashboards para facilitar su comprensión.

**-Conocimiento del negocio:**

Es necesario entender el contexto en el que se aplican los datos para que los resultados sean útiles y relevantes.

## Diferencia entre datos estructurados y no estructurados

### Datos estructurados:

Los datos estructurados son aquellos que están organizados en un formato definido, generalmente en tablas con filas y columnas. Este tipo de datos es fácil de almacenar, consultar y analizar mediante herramientas tradicionales como bases de datos relacionales.

### Características:

- Tienen un formato fijo

- Son fáciles de procesar

- Se almacenan en bases de datos

### Ejemplo:

Una tabla de clientes con campos como nombre, edad, correo electrónico y número de teléfono.

### Datos no estructurados:

Los datos no estructurados no tienen un formato específico ni una organización clara, lo que los hace más complejos de analizar. Representan la mayoría de los datos generados en la actualidad.

### Características:

- No tienen estructura definida

- Son más difíciles de procesar

- Requieren herramientas avanzadas

### Ejemplo:

Imágenes, videos, audios, publicaciones en redes sociales, correos electrónicos o documentos de texto.

| Característica            | Datos Estructurados                                      | Datos No Estructurados                                  |
|--------------------------|----------------------------------------------------------|----------------------------------------------------------|
| Definición               | Datos organizados en un formato fijo (filas y columnas)  | Datos sin una estructura definida                        |
| Organización             | Altamente organizados                                    | Desorganizados o semiestructurados                       |
| Almacenamiento           | Bases de datos relacionales (SQL)                        | Data lakes, NoSQL, almacenamiento en la nube             |
| Facilidad de análisis    | Fácil de analizar                                        | Más difícil de analizar                                  |
| Tipo de formato          | Tablas, hojas de cálculo                                 | Texto, imágenes, audio, video                            |
| Ejemplo                  | Base de datos de clientes                                | Publicaciones en redes sociales                          |
| Procesamiento            | Rápido y sencillo                                        | Requiere herramientas avanzadas                          |
| Flexibilidad             | Baja (estructura rígida)                                 | Alta (estructura flexible)                               |
| Uso común                | Sistemas administrativos y financieros                   | Redes sociales, multimedia, correos electrónicos         |


## Las 5 V del Big Data

El Big Data se caracteriza por cinco elementos fundamentales conocidos como las 5 V, que describen las propiedades de los datos masivos:

### Volumen
Se refiere a la enorme cantidad de datos que se generan constantemente.

### Ejemplo:
Una tienda en línea que registra miles o millones de transacciones diariamente, generando grandes bases de datos.

### Velocidad
Es la rapidez con la que los datos se generan y deben ser procesados.

### Ejemplo:
Las compras en línea o pagos digitales que se realizan en tiempo real y requieren procesamiento inmediato.

### Variedad
Hace referencia a los diferentes tipos de datos que existen.

### Ejemplo:
Datos de texto, imágenes de productos, videos promocionales, opiniones de clientes, etc.

### Veracidad
Se relaciona ocn la calidad y confiabibiliad de los datos.

### Ejemplo:
Eliminiar infomracion duplicada, incorrecta o incompleta para evitar errores en el analisis.

### Valor
Es la capacidad de los datos para generar beneficios o informacion util.

### Ejemplo:

Analizar los datos de clientes para ofrecer recomendaciones personalizadas y aumentar las ventas.
